# Nexus Scientific Calculator

A comprehensive, feature-rich scientific calculator with a beautiful dark theme interface built using modern HTML, CSS, and JavaScript.

## 🌟 Features

### Core Calculator Functions
- **Basic Operations**: Addition, subtraction, multiplication, division
- **Scientific Functions**: Trigonometric (sin, cos, tan), inverse trigonometric (sin⁻¹, cos⁻¹, tan⁻¹)
- **Logarithmic Functions**: Natural log (ln), base-10 log (log), base-2 log
- **Exponential Functions**: e^x, arbitrary exponents
- **Root Functions**: Square root (√), nth root
- **Constants**: π (pi), e (Euler's number)
- **Advanced Operations**: Factorial (!), exponents (^), percentages (%)
- **Parentheses Support**: For complex expressions

### Special Features
- **Quadratic Equation Solver**: Solve equations of the form ax² + bx + c = 0
- **Memory Functions**:
  - MS (Memory Store)
  - MR (Memory Recall)
  - MC (Memory Clear)
  - M+ (Memory Add)
- **Calculation History**: View and review previous calculations
- **Sign Toggle**: ± button to toggle positive/negative
- **Backspace**: Delete last entered character
- **Clear Functions**: AC (All Clear) and CE (Clear Entry)

### User Interface
- **Dark Theme**: Elegant dark interface with gradient accents
- **Glass Morphism Effects**: Modern frosted glass design elements
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Animated Buttons**: Hover effects, click animations, and visual feedback
- **Large Display**: Clear expression and result display
- **Modal History Panel**: Dedicated history viewer with overlay

## 🚀 Usage

### Basic Operations
1. Click number buttons (0-9) to enter values
2. Use operator buttons (+, -, ×, ÷) for calculations
3. Press "=" to compute the result
4. Use parentheses for complex expressions

### Scientific Functions
- Click function buttons (sin, cos, tan, etc.) to insert them
- Enter the value in parentheses or after the function
- Example: `sin(30)` or `sqrt(16)`

### Memory Operations
- **MS**: Store current value in memory
- **MR**: Recall value from memory
- **MC**: Clear memory
- **M+**: Add current value to memory

### History
- Click the "History" button to view previous calculations
- History shows both expressions and results
- Close with the X button, Escape key, or by clicking outside

### Keyboard Shortcuts
- Numbers: `0-9`
- Operators: `+`, `-`, `*`, `/`
- Enter: `=` (equals)
- Backspace: `Backspace` key
- Clear: `Escape` key
- History: `H` key
